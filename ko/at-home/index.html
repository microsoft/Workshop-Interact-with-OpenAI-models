추가 설정 필요

이 워크샵은 집에서 셀프 가이드 세션으로 완료할 수 있습니다. 그러나 설명된 대로 추가 설정 작업과 종속성이 필요합니다.

셀프 가이드 워크샵의 경우 다음이 필요합니다:

  • Azure 구독 → 여기에서 무료 계정 신청.
  • Azure OpenAI에 대한 액세스 → 필요한 경우 여기에서 신청.
  • Azure Open AI 리소스 → Azure OpenAI Studio를 사용하여 만들기
  • gpt-3.5-turbo 모델 배포 → Azure OpenAI Studio를 사용하여 배포

그런 다음 Azure OpenAI Studio로 이동하여 실습을 완료하면 코드 없이 실험할 수 있습니다.

Azure OpenAI Studio: 플레이그라운드 앱
OpenAI 플레이그라운드 앱

OpenAI 플랫폼에는 유효한 OpenAI 계정으로 로그인하면 이 연습을 수행할 수 있는 자체 Playground 애플리케이션이 있습니다. 직접 살펴보는 것을 권장하지만, 워크샵 설명서에서는 Azure OpenAI 사용법에 중점을 두겠습니다.
